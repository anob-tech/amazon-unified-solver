## Extended Description

**English:**  
Amazon Unified Solver is a modular backend project designed to showcase expertise in multiple domains: IT, cybersecurity, web development, logistics, AI, and training.  
Each module is independent but can integrate into a unified system. The project demonstrates enterprise-level repo organization, bilingual documentation, and professional presentation with screenshots.  

Key highlights:  
- **Customer API**: Provides product recommendations and simulates customer interactions.  
- **Logistics Dashboard**: Visualizes supply chain processes and delivery tracking.  
- **Security Module**: Detects scams and malicious activity, enhancing cybersecurity awareness.  
- **AI Services**: Offers text summarization and other lightweight AI utilities.  
- **Training Game**: Educates employees on cybersecurity through interactive quizzes.
